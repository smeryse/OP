# История изменений

## Рефакторинг структуры проекта

### Что изменилось

- ✅ Вся логика перенесена из `2. scripts/script/` в структуру пакета `report_generator/`
- ✅ Создана модульная структура:
  - `report_generator/core/` — основная логика (report.py, image.py, renderer.py, text_processing.py)
  - `report_generator/gui/` — графический интерфейс
  - `report_generator/cli/` — командная строка
  - `report_generator/io/` — ввод-вывод данных
- ✅ Добавлена обработка конфликтов имён при копировании изображений
- ✅ Автоматическая замена «(Рис. N)» → «как показано на Рисунке N»
- ✅ Улучшена обработка путей (автопоиск base_info.json, правильные пути к шаблонам)
- ✅ Обновлён `run.py` для использования новой структуры

### Как использовать

1. **GUI**: `python run.py` (из корня проекта)
2. **CLI**: `python -m report_generator.cli.main --lab 3 -o lab3.html`

### Старые файлы

Файлы в `2. scripts/script/` оставлены для совместимости, но рекомендуется использовать `report_generator/`.
