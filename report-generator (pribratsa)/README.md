# Генератор отчётов по лабораторным работам

Проект для автоматизации написания отчётов: из методички и скриншотов формируется JSON через нейросеть, затем скрипт собирает из него HTML-отчёт по ГОСТу.

##  Инструкция
### 0. Подготовка
  * В файле `3. data/base_prompt.txt` меняем свои данные (И.О. Фамилия)

### 1. Генерация JSON через Deepseek
  * Нейросети скидываем содержание файла `3. data/base_prompt.txt`
  * Также скидываем нейросети лабораторную работу из папки `1. materials/2. labs`

### 2. JSON + картинки → отчёт

* Указываете:
  * путь к **JSON** отчёта по лабе;
  * путь к **base_info.json** (`3. data/base_prompt.txt`);
  * папку с **картинками** (1.png, 2.png …);
  * путь для сохранения **HTML** отчёта (по умолчанию рядом с JSON).

### 3. HTML → печать/сдача

* Открываете HTML в LibreOffice Writer.
* Настраиваете поля по ГОСТ: левое 30мм, верх 20мм, правое 15мм, нижнее 15мм.
* Убираете верхний колонтитул и сохраняете в ODT/DOCX или печатаете.

## Запуск

### GUI

Из корня проекта (`Operating System`):

```bash
python run.py
```

Откроется окно «Генератор отчётов лабораторных работ» с двумя вкладками:

* **🤖 Генерация JSON через AI (В разработке)** — создание JSON из методички.
* **📄 Генерация HTML отчёта** — создание финального отчёта из JSON.

### CLI

```bash
python -m report_generator.cli.main --lab 3 -o lab3.html --images "3. data/lab3/images"
```

**Параметры:**

| Параметр              | Описание                                                             |
| --------------------- | -------------------------------------------------------------------- |
| `--lab <номер>`       | Номер лабораторной работы (ищет JSON в `3. data/lab<N>/lab<N>.json`) |
| `--json <путь>`       | Путь к JSON файлу (альтернатива `--lab`)                             |
| `-o, --output <путь>` | Выходной HTML файл                                                   |
| `--base-info <путь>`  | Путь к base_info.json                                                |
| `--images <путь>`     | Папка с изображениями                                                |
| `--width <px>`        | Максимальная ширина изображений (по умолчанию 500)                   |

## Генерация JSON через AI (В разработке) 🤖

### Поддерживаемые провайдеры

1. **Google Gemini** (рекомендуется)

   * Бесплатно: 1500 запросов/день
   * [Регистрация](https://ai.google.dev/)
   * [API ключ](https://makersuite.google.com/app/apikey)

2. **Groq**

   * Бесплатно: 6000 токенов/мин, 14400 запросов/день
   * [Регистрация](https://console.groq.com/)
   * Очень быстрый ответ

### Использование

1. Откройте вкладку "🤖 Генерация JSON через AI".
2. Выберите файл методички (PDF, TXT, MD).
3. Выберите провайдера (Gemini или Groq).
4. Введите API ключ и нажмите "Сохранить" — ключ сохранится в `.api_keys.json` (не коммитится).

   * Можно использовать переменные окружения (приоритет выше).
5. (Опционально) Укажите номер и тему лабораторной.
6. Нажмите "🚀 Сгенерировать JSON".

> Подробнее см. `report_generator/llm/README.md`

## Установка зависимостей

### Базовые

```bash
pip install pillow jinja2
```

### Для генерации JSON через AI

```bash
# Для Gemini
pip install google-generativeai

# Для Groq
pip install groq

# Для парсинга PDF
pip install PyPDF2
# или альтернативно (лучше работает с таблицами)
pip install pdfplumber
```

Или сразу все зависимости:

```bash
pip install -r requirements.txt
```

> Работает с Python 3.8+
